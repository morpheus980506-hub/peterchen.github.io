这是我正式搭建的 **第一个个人网站**，主题方向是深色科技风，并加入了电子紫的赛博朋克元素。为了完整记录整个过程，我决定写下这篇文章。

---

## 🚀 为什么选择 Hugo？

我希望我的网站：

- **加载速度极快**
- **干净、结构化**
- **可以随时扩展**
- **主题够科技感**

Hugo 属于静态站点生成器，拥有：

- 极致构建速度  
- 强大的模板系统  
- 完美适配 GitHub Pages  

非常适合我这种「喜欢折腾 + 希望可控」的使用者。

---

## 🎨 主题选择：Stack（深色+科技风）

Stack 本身自带优秀的深色风格，我将它 fork 成了自己的版本：

- 主色调改为 **电子紫**
- 部分 UI 采用半透明玻璃效果
- 代码框字体换成 JetBrains Mono

显示效果非常符合我想要的科技氛围。

---

## 🧱 搭建流程（简述）

我主要做了以下步骤：

1. 安装 Hugo（extended）
2. 初始化站点
3. 添加 Stack 主题（Git 子模块）
4. 编写 config.toml
5. 本地预览与调试
6. 使用 GitHub Pages 自动部署
7. 调整自定义 CSS

期间遇到过 Git 识别路径的问题，也已经顺利解决。
